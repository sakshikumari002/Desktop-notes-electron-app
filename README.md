# Desktop Notes App (Workshop App)

This is a cross-platform desktop notes application built using ElectronJS and Antigravity as part of a workshop project.  
Users can add messages/notes in a simple public message stream.

## Live Preview
https://electron-antigravity-kit--sakshikumari920.replit.app

## Features
- Add notes/messages
- Public message stream
- Cross-platform desktop wrapper (Windows, macOS, Linux)

## Tech Stack
- ElectronJS
- Node.js
- React + Vite
- Tailwind CSS
